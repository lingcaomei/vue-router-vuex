# vue-lagou-app

> A Vue.js project 使用vue、vue-router、vuex、vue-resource 仿拉勾网，包含slider滑动组件，刚学习vue练练手~

## Build Setup

``` bash
# 安装依赖
npm install

# 启动 serve with hot reload at localhost:8080
npm run dev

# 编译产出 build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
